# Netflix-data-analysis
Exploratory Data Analysis on Netflix titles dataset using Python, Pandas and matplotlib

## About
This project analyzes Netflix's content library using real 
data from Kaggle. It explores content trends, country 
dominance, genre patterns, and the impact of COVID-19 on 
content addition.

## Questions Answered
1. Which top 10 countries produce the most content?
2. How much content was added per year and why did it drop?
3. Which genre is dominating Netflix?
4. What is the impact of English language on content?
5. Movies vs Shows — which dominates?
6. Which year had the highest/lowest content release?

## Tools Used
- Python
- Pandas (data cleaning + analysis)
- Matplotlib (data visualization)

## Key Findings
- USA dominates with 2808 titles — Hollywood + English = global reach
- Content additions dropped after 2019 — likely due to COVID-19
- Movies outnumber TV Shows 3:1 (6128 vs 2666)
- International Dramas dominate genre category
- India ranks 2nd — Bollywood's massive yearly output
- Non-English countries have lower content due to language barriers

## How To Run
1. Install Jupyter Notebook
2. Download netflix_titles.csv from Kaggle
3. Open the .ipynb file in Jupyter
4. Run all cells top to bottom
